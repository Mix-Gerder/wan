# Installing
You can install WAN dependencies with install.bat or typing `npm install` in CMD.
# Running
Make sure that the configuration file exists (settings.json) then run it with start.bat or typing `npm start` in CMD.
# Dependencies
* node-notifier
* ajax-request
* electron (for UI)
